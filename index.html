CSS selectors cheatsheet

Element, class and ID selectors

Name Tag Description
Element h2 {} Selects all h2 elements on the page
Group h1, h2, h3{} selects all h1, h2 and h3 elements on the page.
Class p {} Selects all elements with class attribute containing foo.
Combining Class p.foo {} Selects all elements with class attribute containing only p elements with foo
ID #foo {} selects the element with 'foo' id attribute value

Contextual selectors

Name Tag Description
Descendant #foo p {} Selects all p elements within the infinite-level hierarchy of element #foo descendants
Adjacent sibling h2 + p {} Selects the sibling element p that is immediately next to h2 element
Child #foo > p {} Selects all elements p that are siblings to the h2 element
General sibling h2 ~ p {} Selects all elements p that are siblings to the h2 element

Pseudo-class selectors

Pseudo-class selectors for link and user states

Name Tag Description
Unvisited link a:link {} Applies to link elements that have not been visited
Visited link a:visited {} Applies to link elements that have been visited
Focus state .foo:focus {} Applies to selected .foo element that is ready for input
Hover state .foo:hover {} Applies when mouse pointer is over the .foo element
Active state .foo:active {} Applies when .foo element is in process of being clicked

Pseudo-class selectors that apply to siblings

First child .foo:first-child {} Selects the specified .foo element when it is the first child of its parent
Last child .foo:last-child {} Selects the specified .foo element when it is the last child of its parent
Only child .foo:only-child {} Selects the specified .foo element when it is the only child of its parent
First of type h2:first-of-type {} Selects the h2 element when it is the first element of its type within its parent element
Last of type h2:last-of-type {} Selects the h2 element when it is the last element of its type within its parent element
Only of type h2:only-of-type {} Selects the h2 element when it is the only element of its type within its parent element
Nth child .foo:nth-child(n) {} Selects the nth .foo child element
Nth last child h2:nth-of-type(n) {} Selects the nth h2 child element of its type
Nth last of type h2:nth-last-of-type(n) {} Selects the nth h2 child element of its type counting backwards
