CSS selectors cheatsheet

Element, class and ID selectors

Name Tag Description
Element h2 {} Selects all h2 elements on the page
Group h1, h2, h3{} selects all h1, h2 and h3 elements on the page.
Class p {} Selects all elements with class attribute containing foo.
Combining Class p.foo {} Selects all elements with class attribute containing only p elements with foo
ID #foo {} selects the element with 'foo' id attribute value

Contextual selectors

Name Tag Description
Descendant #foo p {} Selects all p elements within the infinite-level hierarchy of element #foo descendants
Adjacent sibling h2 + p {} Selects the sibling element p that is immediately next to h2 element
Child #foo > p {} Selects all elements p that are siblings to the h2 element
General sibling h2 ~ p {} Selects all elements p that are siblings to the h2 element
